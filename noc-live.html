NOC live dashboard (xray-core gallery · startPolling)

Wire xrayCore.startPolling() to your live telemetry (SNMP/gNMI/API) and the topology updates itself. Here a simulated feed flaps the r1–r3 path; the view follows in real time — no page reload, no manual clicks.

poll interval: 2.5s
— 
// real usage: poll your own state endpoint, the view auto-updates
var view = xrayCore.renderTopology('#topo', CONFIG, { topology, trace });
view.startPolling(() => fetch('/api/state').then(r => r.json()), 3000);

// the fetcher may return a bare state, or a full snapshot:
//   { state, topology, trace }  ← overview links/arrows update too
