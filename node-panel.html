X-Ray — single-node panel

Position-independent control-plane view for ONE node (Routing + BGP + Best-Path). Drop it into any topology GUI's node panel — pick a node, pass its clab-collect.js state, done.

Wiring is one line — XrayNodePanel.render(document.getElementById('xray-panel'), nodeState).
No graph, no coordinates: identical output wherever the node sits in the topology.
Colours are CSS variables — override --xnp-* on .xnp-root (the Theme picker above just sets those). In VS Code, map them to theme vars, e.g. --xnp-bg: var(--vscode-editor-background).
