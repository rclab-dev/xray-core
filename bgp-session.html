BGP: eBGP session establishment (xray-core gallery)

ルータを起動せずに eBGP セッションが Idle → Established に至る FSMを見せる (R1 AS 65001 — R2 AS 65002)。各ボタンで状態を切替。「Open the router」 で円柱を開くと、BGP プロセッサとセッション(トンネル)が中で見える。Established で r1 が広告する 203.0.113.0/24 を学習。

① TCP :179 接続
② OPEN / capabilities 交換
③ Established(KEEPALIVE / UPDATE)
// load: xray-core.js (engine) + xray-api.js (facade) + bgp-data.js
var view = xrayCore.renderTopology('#topo', BGP_CONFIG, { topology: TOPO });

// each state button = one snapshot
view.applyState(BGP_STATES.opensent);      // → OPEN 交換中 (まだ未確立)
view.applyState(BGP_STATES.established);    // → トンネル紫 + 203.0.113.0/24 を学習

// "inside the router" cylinder:
view.openDeepDive();
Field shapes: see DATA-CONTRACT.md. Descriptive only — it visualizes BGP session state, it is not a simulator. The engine renders BGP as is_established (Established → purple session tunnel) and installs the learned prefix from route_resolution; the Idle→ Established progression above is shown by the 3-step indicator (the engine itself draws only the Established / not-Established distinction).
The live ping/packet animation is hidden here (this demo is about the session, not traffic).
