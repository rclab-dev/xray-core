X-Ray — topology overlay

The whole graph as a live control-plane picture: links coloured by adjacency (OSPF Full / BGP Established), and a forwarding-path trace that follows each router's own routing table hop by hop. Pick a destination, click a node to trace from it.

Wiring is one line — XrayTopoOverlay.render(document.getElementById('xray-topo'), { states, positions }).
states = clab-xray-collect.js's window.LIVE_STATES; positions = node coords from the topology JSON / TopoViewer annotations.
Colours are CSS variables (--xto-* on .xto-root) — same theming hook as the node panel. Click a node = trace source · pick a destination = trace target.
